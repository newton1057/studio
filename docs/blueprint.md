# **App Name**: Ima Health Navigator

## Core Features:

- Pathway Selection: Allow users to select from predefined health pathways based on their goals (e.g., 'start taking care of my health,' 'have a sports goal,' 'have a diagnosis')
- Pathway Content Display: Display content related to the selected pathway, which could include articles, tips, guides, and tools.
- Responsive Design: Ensure the application is fully responsive and adapts to different screen sizes and devices.

## Style Guidelines:

- Background color: Dark gray (#12181B), providing a sleek and modern feel, similar to the original image.
- Primary color: Light lime green (#D8E9A8), used for accents, buttons, and interactive elements to draw attention and create a sense of vibrancy.
- Accent color: Soft White (#F5F5F5), used for text and other UI elements over the dark background.
- Body and headline font: 'Inter', sans-serif, for a clean, modern and readable user interface.
- Use a card-based layout with rounded corners to present each health pathway as a distinct section, consistent with the reference image.
- Employ simple, outlined icons to represent each health pathway, enhancing visual appeal and ease of understanding.
- Incorporate subtle hover effects on pathway cards and button elements to provide visual feedback and enhance interactivity.
